===============================================
            背景图设置完整指南
===============================================

您的背景图没有显示的原因可能有以下几种，请按步骤检查：

📁 背景资源位置
====================
发现您的5层背景图位于：
Assets/Map/2 Background/Day/
- 1.png (远景天空)
- 2.png (远山)  
- 3.png (中景)
- 4.png (近景树木)
- 5.png (前景装饰)

🔧 设置步骤（请严格按顺序执行）
========================================

步骤1：在场景中创建背景对象
--------------------------------
1. 在Hierarchy面板右键 → Create Empty
2. 重命名为"BackgroundParallax"
3. 在BackgroundParallax下再创建5个Empty子对象：
   - Background_Layer1 (远景)
   - Background_Layer2 (远山)
   - Background_Layer3 (中景)
   - Background_Layer4 (近景)
   - Background_Layer5 (前景)

步骤2：为每个图层添加Sprite Renderer
-------------------------------------
为每个Background_Layer对象添加Sprite Renderer组件：
1. 选中Background_Layer1
2. Add Component → Rendering → Sprite Renderer
3. 将Assets/Map/2 Background/Day/1.png拖到Sprite字段
4. 重复这个步骤为Layer2-5设置2.png-5.png

步骤3：设置图层顺序和位置
--------------------------
重要！设置正确的Sorting Layer和Order：
- Background_Layer1: Order in Layer = -5
- Background_Layer2: Order in Layer = -4  
- Background_Layer3: Order in Layer = -3
- Background_Layer4: Order in Layer = -2
- Background_Layer5: Order in Layer = -1

位置设置：
- 所有图层的Z坐标设为10（远离摄像机）
- X和Y坐标根据需要调整对齐

步骤4：添加ParallaxController脚本
-----------------------------------
1. 选中BackgroundParallax对象
2. Add Component → 搜索"ParallaxController"
3. 如果没有，请确保Assets/Scripts/ParallaxController.cs存在

步骤5：配置视差系数
--------------------
在ParallaxController组件中：
1. Reference Camera: 拖入Main Camera
2. 点击"Parallax Layers"旁的箭头，设置Size为5
3. 为每个Element设置：

   Element 0 (远景天空):
   - Layer Transform: Background_Layer1
   - Parallax Factor: 0.1
   - Layer Name: "远景天空"
   - Enabled: ✓

   Element 1 (远山):
   - Layer Transform: Background_Layer2
   - Parallax Factor: 0.3
   - Layer Name: "远山"
   - Enabled: ✓

   Element 2 (中景):
   - Layer Transform: Background_Layer3
   - Parallax Factor: 0.5
   - Layer Name: "中景"
   - Enabled: ✓

   Element 3 (近景):
   - Layer Transform: Background_Layer4
   - Parallax Factor: 0.7
   - Layer Name: "近景树木"
   - Enabled: ✓

   Element 4 (前景):
   - Layer Transform: Background_Layer5
   - Parallax Factor: 0.9
   - Layer Name: "前景装饰"
   - Enabled: ✓

步骤6：在编辑器中使用快捷功能
------------------------------
选中BackgroundParallax对象，在Inspector中：
1. 点击"应用5层背景推荐设置"按钮
2. 点击"显示图层信息"检查配置

🚨 常见问题排查
===================

问题1：背景图不显示
-----------------
检查项：
- Sprite Renderer的Sprite字段是否已设置
- Order in Layer是否为负数（-5到-1）
- 图层的Z坐标是否合适（建议10）
- 摄像机的Clear Flags是否为Skybox或Solid Color

问题2：背景不跟随摄像机移动
-------------------------
检查项：
- ParallaxController的Enable Parallax是否勾选
- Reference Camera是否设置为Main Camera
- 各图层的Layer Transform是否正确拖入
- Parallax Factor是否大于0

问题3：背景移动过快或过慢
-----------------------
调整Parallax Factor：
- 0 = 完全不动（静态背景）
- 0.1-0.3 = 很慢（远景）
- 0.5-0.7 = 中速（中景）
- 0.8-1.0 = 快速（近景）

问题4：背景层级错误
-----------------
确保Order in Layer从远到近递增：
远景(-5) < 远山(-4) < 中景(-3) < 近景(-2) < 前景(-1) < 地面(0+)

🎯 快速验证
==============
1. 在Game视图中移动摄像机
2. 背景应该以不同速度跟随移动
3. 远景移动最慢，前景移动最快
4. 控制台应该显示绿色初始化信息

📝 调试信息
=============
如果仍有问题，请查看控制台：
- 绿色信息：正常工作
- 黄色信息：警告
- 红色信息：错误，需要修复

在ParallaxController中启用"Show Debug Info"可以看到详细信息。

🔄 重置功能
==============
如果设置混乱，可以：
1. 选中BackgroundParallax对象
2. 右键点击ParallaxController → "重置所有图层位置"
3. 或在Inspector中点击"重置所有图层位置"按钮

设置完成后，您应该能看到美丽的5层视差滚动背景！
