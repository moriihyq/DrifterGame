🎯 极安全烛火系统 - 超简单5步安装指南
=================================================================

⚡ 最快方式（2分钟完成）
=================================================================

第1步：创建烛火对象
──────────────────────────
1. 在Hierarchy中右键 → Create Empty
2. 命名为 "SafeCandle"
3. 设置位置到合适的游戏场景位置

第2步：一键自动设置
──────────────────────────
1. 将 UltraSafeQuickSetupWizard.cs 拖到烛火对象上
2. 在Inspector中点击右上角三点菜单
3. 选择 "创建完整烛火系统"
4. 等待2秒，自动创建完成！

第3步：验证安装
──────────────────────────
1. Inspector会自动显示验证器组件
2. 点击右上角三点菜单 → "执行系统验证"
3. 查看Console，确认显示"✅ 系统验证通过"

第4步：测试功能
──────────────────────────
1. 运行游戏
2. 控制玩家靠近烛火（2.5米内）
3. 看到升级提示后按E键
4. 确认升级效果和烛火熄灭

第5步：完成！
──────────────────────────
🎉 恭喜！您的烛火交互系统已经可以正常工作了！

🔧 手动安装方式（3分钟完成）
=================================================================

如果喜欢手动控制每个步骤：

第1步：添加核心脚本
──────────────────────────
1. 创建空物体命名为"SafeCandle"
2. 拖拽 UltraSafeCandleSystem.cs 到物体上
3. 拖拽 UltraSafeCandleIdentifier.cs 到物体上

第2步：基础配置
──────────────────────────
在UltraSafeCandleSystem组件中设置：
• Current Candle Level: 1
• Interaction Range: 2.5
• Interaction Key: E
• Auto Create UI: ✓
• Enable Safe Debug Log: ✓

第3步：添加验证器（推荐）
──────────────────────────
1. 拖拽 UltraSafeInstallationValidator.cs 到物体上
2. 运行验证检查所有配置

第4步：测试运行
──────────────────────────
运行游戏并测试交互功能

=================================================================

• 材质变为灰色
• 播放升级音效

🔍 故障排除
=================================================================

问题：升级提示不显示
解决：确认玩家标签为"Player"，检查交互距离

问题：按E键无反应
解决：确认玩家在2.5米范围内，查看Gizmos范围显示

问题：踢腿伤害未修改
解决：确认PlayerAttackSystem组件存在且有KickDamage属性

问题：动画不播放
解决：确认玩家有Animator组件和windpower触发器

万能解决方案：
运行验证器 → 查看Console日志 → 按照提示修复
